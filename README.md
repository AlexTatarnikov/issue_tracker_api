# Issue Tracker API

The idea behind the task is to implement an issue tracker like JIRA, Zendesk, or Trello. It should be very simple with only few feature.

#### Requirements:

* Ruby/Rails versions
    * ruby ~> '2.4.2'
    * rails ~> '5.1.4'

* System dependencies
    * postgresql 9.6

#### Application init

* `bin/setup`

#### HowTos

* How to run the test suite: `bundle exec rspec`

*Version 11 November 2017*
